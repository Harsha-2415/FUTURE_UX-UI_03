# 📚 Wisdom Centre - E-Learning Mobile App UI/UX

**Wisdom Centre** is a modern **E-Learning Mobile Application** designed in **Figma**. The project focuses on delivering a clean, engaging, and user-friendly learning experience where users can explore courses, enroll in programs, and continue learning seamlessly.

---

## 📱 Screens Included

- Splash Screen
- Login / Sign In
- Home Dashboard
- Course Categories
- Popular Courses
- Course Details
- Enrollment Success

---

## ✨ Features

- Modern & Minimal UI Design
- Clean User Experience
- Secure Login Screen
- Search Courses
- Browse Learning Categories
- Popular Courses Section
- Continue Learning Progress
- Course Details
- One-Tap Course Enrollment
- Enrollment Success Screen
- Responsive Mobile Design
- Easy Navigation

---

## 🎨 Tools Used

- Figma
- Auto Layout
- Components
- Variants
- Material Icons
- Google Fonts
